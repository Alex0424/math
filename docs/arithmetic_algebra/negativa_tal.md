# Negativa Tal

Negativa tal tillhör de hela talen $\mathbb{Z}$.

## Addition och Subtraktion

$$a + (-b) = a - b$$

$$a - (-b) = a + b$$

## Teckentabell (Multiplikation och Division)

| Tecken | Resultat |
|--------|----------|
| $+ \;\cdot\; +$ | $+$ |
| $+ \;\cdot\; -$ | $-$ |
| $- \;\cdot\; +$ | $-$ |
| $- \;\cdot\; -$ | $+$ |

Ett **jämnt** antal minustecken $(-x)^4$ ger positivt resultat; ett **udda** antal $(-x)^3$ ger negativt.

## Multiplikation

$$a \cdot (-b) = (-a) \cdot b = -ab$$

$$(-a) \cdot (-b) = ab$$

## Division

$$\frac{-a}{b} = \frac{a}{-b} = -\frac{a}{b}$$

$$\frac{-a}{-b} = \frac{a}{b}$$
