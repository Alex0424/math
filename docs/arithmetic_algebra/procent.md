# Procent och Förändringsfaktorn

## Andelen, Delen, Det Hela

Andelen = Delen / det hela

Delen = Andelen * Det hela

Det Hela = Delen / Andelen

## Förändringsfaktor

Nya värdet = Förändringsfaktorn * Gamla värdet

Förändringsfaktorn = Nya värdet / Gamla värdet

## Flera procentuella förändringar

Nya värdet = Totala förändringsfaktorn * Gamla värdet
